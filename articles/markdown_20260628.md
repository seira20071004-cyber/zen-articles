こんにちは！文章の構造化や装飾を手軽に行える軽量マークアップ言語「マークダウン（Markdown）」。
これさえ覚えておけば、ブログ、メモ、仕様書などのドキュメント作成が圧倒的にスムーズになります。

今回は、日常的に使う基本的な書き方から、知っておくと便利な応用的な書き方までを分かりやすくまとめました。

---

## 1. 導入（はじめに）

### マークダウンとは？
マークダウン（Markdown）とは、特別なツールを使わずに「普通の文字（プレーンテキスト）」のままで、見出しや箇条書き、リンクなどの構造化を手軽に行える軽量マークアップ言語です。

### なぜ使うのか（メリット）
- **HTMLのような複雑なタグを書く必要がない**：`<h1>` や `<ul>` などのタグを入力する手間が省けます。
- **執筆・メモのスピードが上がる**：キーボードだけで素早く装飾ができるため、タイピングの手を止める必要がありません。
- **多くのモダンなツールで標準採用されている**：GitHub、Notion、各種ブログ（QiitaやZennなど）でそのまま使えるため、一度覚えれば一生モノのスキルになります。

---

## 2. 基本的な書き方（チートシート）

マークダウンのメインとなる基本的な書き方の一覧です。実際の記述例と表示結果のイメージを並べてご紹介します。

| 機能 | 記述例 | 表示結果（イメージ） |
| :--- | :--- | :--- |
| **見出し** | `# 見出し1`<br>`## 見出し2` | **大きな文字**、**中くらいの文字** |
| **太字** | `**太字にする文字**` | **太字にする文字** |
| **箇条書き** | `* 項目1`<br>`* 項目2` | • 項目1<br>• 項目2 |
| **番号付きリスト** | `1. 最初の項目`<br>`2. 次の項目` | 1. 最初の項目<br>2. 次の項目 |
| **リンク** | `[Google](https://google.com)` | [Google](https://google.com) （クリック可能） |
| **画像の挿入** | `![代替テキスト](画像のURL)` | （指定した画像が表示される） |
| **引用** | `> ここは引用文です。` | 枠で囲まれた引用文のスタイルになります |

## 3. エンジニア・ライターに嬉しい便利な書き方

基本に加えて、以下の機能も知っておくと記事の読みやすさと価値がグッと上がります。

### コードの挿入（コードブロック）
バッククォート3つ（ \`\`\` ）で囲むことで、プログラムのコードを綺麗に色分け（シンタックスハイライト）して表示できます。始まりの3つの後ろに `python` や `javascript` などの言語名を指定します。
````markdown
```python
print("Hello, World!")
```
````
### 表（テーブル）の作成
縦棒（ `|` ）とハイフン（ `-` ）を使うことで、簡単に表を作ることができます。2行目のコロン（ `:` ）の位置で、左寄せ・中央寄せ・右寄せを調節できます。

```markdown
| カテゴリ | 難易度 |
| :--- | :---: |
| 基本情報 | 初級 |
| 応用情報 | 中級 |
```

### チェックボックス（タスクリスト）
`- [ ]` （未完了）や `- [x]` （完了）と書くことで、タスクリストを作成できます。※括弧の中のスペースは半角です。

```markdown
- [ ] 未完了のタスク
- [x] 完了したタスク
```

## 4. まとめ（おわりに）
